# SQL-Employee-Tracker

## Description

The SQL Employee Tracker is a content management system (CMS) interface that allows non-developers to easily view and interact with their staff's information. This is a command-line application that uses Node.js, Inquirer, and a MySQL database. 

### User Story

```md
AS A business owner
I WANT to be able to view and manage the departments, roles, and employees in my company
SO THAT I can organize and plan my business
```
